## Overview

Learn how to implement EKIO's complete visual identity system in your R visualizations. This tutorial covers the Modern Premium theme, color scales, typography, and professional styling techniques.

::: {.callout-tip}
## Prerequisites
- Basic ggplot2 knowledge
- R and RStudio installed
- Understanding of data visualization principles
:::

## Learning Objectives

By the end of this tutorial, you will be able to:

1. Install and configure EKIO's visual identity system
2. Apply EKIO themes to your ggplot2 charts
3. Use EKIO color scales for different data types
4. Create professional, branded visualizations
5. Export charts with consistent styling

## Setup

First, let's load the required packages and set up our environment:

```{r setup}
#| message: false
#| warning: false

# Load required packages
library(tidyverse)
library(scales)
library(extrafont)
library(geobr)

# Source EKIO theme functions
source("https://raw.githubusercontent.com/ekio-brazil/ekio-r-theme/main/ekio_theme.R")

# Set global theme
theme_set(theme_ekio())
```

## EKIO Color System

The EKIO Modern Premium theme uses a sophisticated blue-based color palette:

```{r color-system}
# EKIO color definitions
ekio_colors <- list(
  primary = "#2C6BB3",
  secondary = "#1abc9c",
  accent = c("#f39c12", "#e74c3c", "#9b59b6"),
  sequential = c("#f4f7fb", "#e3ecf5", "#d2e1ef", "#a0bfdb", 
                 "#6d9dc7", "#2C6BB3", "#265c9a", "#204d81", "#1a3e68"),
  categorical = c("#2C6BB3", "#1abc9c", "#f39c12", "#e74c3c", "#9b59b6")
)

# Display color palette
tibble(
  color = ekio_colors$sequential,
  value = 1:length(ekio_colors$sequential)
) %>%
  ggplot(aes(x = value, y = 1, fill = color)) +
  geom_col(width = 1) +
  scale_fill_identity() +
  theme_void() +
  labs(title = "EKIO Sapphire Sequential Scale")
```

## Basic Chart with EKIO Theme

Let's create a simple chart using Brazilian economic data:

```{r basic-chart}
# Sample Brazilian GDP data
gdp_data <- tibble(
  year = 2015:2024,
  gdp_growth = c(-3.5, -3.3, 1.3, 1.8, 1.2, -3.9, 4.2, 2.9, 3.1, 2.8),
  region = "Brazil"
)

# Create chart with EKIO theme
gdp_chart <- gdp_data %>%
  ggplot(aes(x = year, y = gdp_growth)) +
  geom_line(size = 1.2, color = ekio_colors$primary) +
  geom_point(size = 3, color = ekio_colors$primary) +
  geom_hline(yintercept = 0, linetype = "dashed", alpha = 0.6) +
  scale_y_continuous(labels = percent_format(scale = 1)) +
  labs(
    title = "Brazilian GDP Growth Rate",
    subtitle = "Annual percentage change in real GDP, 2015-2024",
    x = "Year",
    y = "GDP Growth Rate (%)",
    caption = "Source: IBGE (2024) | EKIO Analytics"
  ) +
  theme_ekio()

gdp_chart
```

## Choropleth Map with EKIO Colors

Now let's create a professional choropleth map using Brazilian state data:

```{r choropleth-map}
#| fig-width: 10
#| fig-height: 8

# Load Brazilian states
states <- read_state(year = 2020, showProgress = FALSE)

# Sample economic data for states
set.seed(123)
state_data <- states %>%
  select(code_state, name_state) %>%
  st_drop_geometry() %>%
  mutate(
    gdp_index = runif(n(), min = 60, max = 140),
    gdp_category = case_when(
      gdp_index < 80 ~ "Low",
      gdp_index < 100 ~ "Medium-Low", 
      gdp_index < 120 ~ "Medium-High",
      TRUE ~ "High"
    )
  )

# Create choropleth map
brazil_map <- states %>%
  left_join(state_data, by = "code_state") %>%
  ggplot() +
  geom_sf(aes(fill = gdp_index), color = "white", size = 0.2) +
  scale_fill_gradientn(
    colors = ekio_colors$sequential,
    name = "GDP Index",
    guide = guide_colorbar(
      title.position = "top",
      barwidth = 15,
      barheight = 0.8
    )
  ) +
  theme_ekio() +
  theme(
    axis.text = element_blank(),
    axis.ticks = element_blank(),
    panel.grid = element_blank(),
    legend.position = "bottom"
  ) +
  labs(
    title = "Brazilian States Economic Performance Index",
    subtitle = "Relative economic development indicators by state",
    caption = "Source: IBGE Regional Accounts | EKIO Analytics"
  )

brazil_map
```

## Multiple Series Chart

For categorical data, use EKIO's categorical color palette:

```{r multiple-series}
# Sample sectoral data
sector_data <- expand_grid(
  year = 2020:2024,
  sector = c("Agriculture", "Industry", "Services")
) %>%
  mutate(
    employment_growth = case_when(
      sector == "Agriculture" ~ c(2.1, -1.2, 3.4, 1.8, 2.3),
      sector == "Industry" ~ c(-8.7, 4.5, 2.1, 3.2, 1.9),
      sector == "Services" ~ c(-7.8, 8.2, 4.1, 2.8, 3.1)
    )
  )

# Create multi-series chart
sector_chart <- sector_data %>%
  ggplot(aes(x = year, y = employment_growth, color = sector)) +
  geom_line(size = 1.2) +
  geom_point(size = 3) +
  scale_color_manual(values = ekio_colors$categorical[1:3]) +
  scale_y_continuous(labels = percent_format(scale = 1)) +
  labs(
    title = "Employment Growth by Economic Sector",
    subtitle = "Annual percentage change in employment, Brazil 2020-2024",
    x = "Year",
    y = "Employment Growth (%)",
    color = "Sector",
    caption = "Source: IBGE, PNAD Cont√≠nua | EKIO Analytics"
  ) +
  theme_ekio()

sector_chart
```

# language-switch.js
function switchLanguage(lang) {
  const currentUrl = window.location.pathname;
  let newUrl;
  
  if (lang === 'pt') {
    if (currentUrl.startsWith('/en/')) {
      newUrl = currentUrl.replace('/en/', '/pt/');
    } else {
      newUrl = '/pt' + currentUrl;
    }
  } else {
    if (currentUrl.startsWith('/pt/')) {
      newUrl = currentUrl.replace('/pt/', '/en/');
    } else {
      newUrl = '/en' + currentUrl;
    }
  }
  
  window.location.href = newUrl;
}

// Auto-detect browser language on first visit
document.addEventListener('DOMContentLoaded', function() {
  const hasVisited = localStorage.getItem('ekio-academy-visited');
  
  if (!hasVisited) {
    const browserLang = navigator.language.slice(0, 2);
    if (browserLang === 'pt' && !window.location.pathname.startsWith('/pt/')) {
      switchLanguage('pt');
    }
    localStorage.setItem('ekio-academy-visited', 'true');
  }
});

## Content Management

### Adding New Tutorial

1. Create folder: `tutorials/new-tutorial/`
2. Add `index.qmd` with YAML header:
   ```yaml
   ---
   title: "Tutorial Title"
   author: "EKIO Academy"
   date: "2024-03-15"
   categories: [tag1, tag2]
   ---
   ```
3. Update `tutorials/index.qmd` with link

### Bilingual Content

- English content in root folders
- Portuguese content in `pt/` folders
- Use language switching JavaScript for navigation
- Maintain parallel structure

### EKIO Branding

- Use EKIO color palette in `custom.scss`
- Apply EKIO themes to R code chunks
- Include EKIO watermarks in visualizations
- Add professional CTA sections
